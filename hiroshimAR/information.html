“HiroshimARchive”について
About “HiroshimARchive”

“HiroshimARchive”は，広島原爆の実相をつたえるデジタルアーカイブ「ヒロシマ・アーカイブ」の収蔵資料を，地図とAR（拡張現実）で閲覧できるアプリです．

“HiroshimARchive” is an iPhone Apps which indicates the data of “Hiroshima Archive” that is a pluralistic digital archive of the Hiroshima atmic bomb by GPS map and AR technology.

地図とAR表示をもちいて，実際の広島の街を歩きながら，地図や街並みに資料を重ねあわせ，1945年に投下された原爆についての理解を深めることができます．

By using “Map mode”, Data are displayed by touching markers. And more, in Hiroshima, Users can know the situation of Hiroshima in 1945 using a GPS function. By using “The AR (Augmented Reality) mode”, Data can be displayed by AR overlapping with actual cityscape.

Twitter投稿にも対応しています。ハッシュタグ「#hiroshima0806」とジオタグを付けて投稿されます。

“HiroshimARchive” has a function of Twitter. The hashtag “#hiroshima0806” is attached to user's tweets. Communication between users is urged by this function.

“HiroshimARchive”アプリに関する詳しい情報は，公式ページの「ARアプリ」をご覧ください．

Please refer to the official website for getting more detailed information.
